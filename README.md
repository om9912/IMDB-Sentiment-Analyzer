# IMDB Movie Review Sentiment Analysis

A Deep Learning-powered web application that classifies movie reviews as Positive or Negative using a trained Simple RNN (Recurrent Neural Network) model and an interactive Streamlit user interface.

# Project Overview

This project performs sentiment analysis on movie reviews from the IMDB dataset. Users can enter a movie review through a Streamlit web application, and the trained RNN model predicts whether the review expresses a positive or negative sentiment.

The application demonstrates the use of:

Natural Language Processing (NLP)
Deep Learning with TensorFlow/Keras
Recurrent Neural Networks (RNN)
Streamlit for Web Deployment
1. Features
2. Real-time sentiment prediction
3. Pre-trained Simple RNN model
4. Confidence score display

# Installation
Clone Repository
git clone https://github.com/your-username/imdb-sentiment-analysis.git

cd imdb-sentiment-analysis
Create Virtual Environment
python -m venv venv
Activate Environment
Windows
venv\Scripts\activate
Linux/Mac
source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
# Run the Application

Start the Streamlit server:

streamlit run app.py

The application will open automatically in your browser.

Default URL:

http://localhost:8501
